Blogging with Pelican

Sat 18 January 2014
By Jestin Joy

After a brief stop, decided to continue with blogging again with pelican.

This is how to do it.

  1. Create an account in Github pages. Just create the repo. Don't follow the other steps.

  2. Install pelican. I have used pip.

    pip install pelican

  3. Use the pelican-quickstart command to insert default settings. This will ask for some questions. Answer them.

  4. Edit the contents of pelicanconf.py to add some more contents

  5. Use the make file to push the contents to Github.

    make github
Proudly powered by Pelican, which takes great advantage of Python.
