Introduction to computing and problem solving using Python

Fri 19 June 2015
By Jestin Joy

Taking a cue from MIT, Kerala Technological University has plans for teaching Python as an introductory course for BTech Computer Science students. The first draft published has topics

___

Introduction to Python – variables, expressions and statements, evaluation of expressions, precedence, string operations. Functions, calling functions, type conversion and coercion, composition of functions, mathematical functions, user-defined functions, parameters and arguments

Control statements, Boolean expressions and logical operators, conditional and alternative executions. Iteration - while statement and tables.

Strings and lists – string traversal and comparison with examples. List operations with examples. tuples and dictionaries – operations and example

Files and exceptions – text files, directories. Introduction to classes and objects – attributes, instances ___
