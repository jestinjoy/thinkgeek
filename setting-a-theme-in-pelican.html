Setting a theme in Pelican

Date Fri 31 July 2015 By Jestin Joy Category general. Tags pelican blog

This explains how to set a new theme for your blog. Pelican comes with a command called pelican-themes. Current themes can be listed with the command

pelican-themes -l

New theme can be installed with the command

pelican-themes --install ~/pelican-themes/bootstrap2

where ~/pelican-themes/bootstrap2 represents the downloaded theme. Themes can be downloaded from the pelican theme Github.
