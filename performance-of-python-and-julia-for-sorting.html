Performance of Python and Julia for sorting

Date Thu 20 February 2014 Tags python / pelican / sorting / performance

With Wired terming it is as "a new language they hope will be suited to practically any task. ", thought of giving it a try. One major feature emphasized about Julia was its speed. So run sorting on 1,00,000 numbers using both Julia and Python. What I have done is

  1. Run inbuilt sorting methods of Julia and Python on large set of numbers
  2. Run custom sorting method (insertion sort based on programming pearls book) using Julia and Python

As expected Julia completed sorting faster than Python for inbuilt and custom versions.

I have used matplotlib to plot the result.
